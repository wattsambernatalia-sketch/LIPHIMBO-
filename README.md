# LIPHIMBO

A community-powered app that turns local problems into verifiable micro-jobs and real-world action.

## Core Idea
Users report problems → others accept tasks → proof is submitted → community verifies → impact is tracked.

## MVP Features
- User accounts
- Report issues
- Accept tasks
- Upload proof
- Verification system

## Tech Direction
Mobile-first application (Flutter planned)
Backend (Firebase or alternative planned)
